# speech-and-speaker-recognition-system
- 基于第三方接口实现语音识别和语音合成、说话人识别功能

## requirements
```
pip install matplotlib==3.0
pip install keras==2.1
pip install tensorflow==1.8
pip install flask
pip install scikit-learn==0.19
```
## run
```
python manage.py
```
